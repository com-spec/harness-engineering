# 機能要求: マルチターン会話履歴ビュー（3バリアント共通・一字一句同一で渡す）

対象リポジトリ: ローカルナレッジベースアプリ（TypeScript / React / Electron）
対象: `src/renderer/components/ConversationHistory.tsx`（現在はプレースホルダ実装）

## 背景

このアプリのQ&Aは、取り込んだドキュメントを根拠に、質問へ出典（citations）つきの回答を返す。
現在の履歴表示は簡易リストで、回答は途中で切れ、出典も表示されない。

## 要求

履歴表示を本格的な「会話ビュー」に仕上げてほしい。

1. ユーザーが過去のやり取り（質問・回答・出典）を会話として読み返せること
2. 回答全文と出典を確認できること
3. アプリ利用の流れの中で自然に使えること（やり取りの確認、続けて質問する、など）
4. 見た目・操作性も製品品質を目指すこと

## 完了条件

- `npm run check` と `npm run check:arch` が成功すること
- テストを追加した場合は `npm test` も成功すること
- 既存のアーキテクチャ境界（`docs/ARCHITECTURE.md`）を守ること
- 作業はカレントブランチにコミットすること
